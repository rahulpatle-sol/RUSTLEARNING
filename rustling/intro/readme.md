
# Rust Language Introduction & Exercises

## Overview

This project contains introductory Rust concepts and practical exercises to help you learn the Rust programming language.

## Contents

- **Variables & Mutability** - Understanding `let`, `mut`, and variable binding
- **Data Types** - Integers, floats, booleans, and characters
- **Functions** - Declaring and calling functions
- **Control Flow** - `if` statements, loops, and pattern matching
- **Ownership** - Borrowing, references, and lifetimes
- **Structs & Enums** - Custom data types
- **Error Handling** - `Result` and `Option` types

## Getting Started

### Prerequisites

- [Rust](https://www.rust-lang.org/tools/install) installed

### Running Exercises

```bash
cargo run --example <exercise_name>
```
